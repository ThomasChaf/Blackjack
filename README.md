# Blackjack
A black jack simulator

## Game environement

- Basic strategy
- HI-LO count
- 100% of penetration in the shoe.
- Double after split : not allowed
- Soft 17 : Dealer hits
- Surrender : Allowed
- 
## Configurable

- Nb deck (default 6)
- Initial bankroll (default 10000)
- Single mise (1/500 of the bankroll)
